I installed the [Hugo Language and Syntax Support](https://github.com/theNewDynamic/language-hugo-vscode) plugin. It adds syntax highlighting for Hugo code in HTML, but not much else. I also tried [Hugo Helper](https://code.visualstudio.com/Docs/editor/tasks#_processing-task-output-with-problem-matchers) for automatically generating content, but it turned out bash scripts were easier.
