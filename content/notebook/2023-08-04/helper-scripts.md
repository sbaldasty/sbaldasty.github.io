I wrote scripts to help with efficiency of website-related tasks: [ud-np.sh](https://github.com/sbaldasty/sbaldasty.github.io/blob/main/bin/ud-np.sh) to create a notebook page for the current date, [ud-n.sh](https://github.com/sbaldasty/sbaldasty.github.io/blob/main/bin/ud-n.sh) to add a note to the notebook page for the current date - it accepts a parameter for the base of the filename, and [ud-serve.sh](https://github.com/sbaldasty/sbaldasty.github.io/blob/main/bin/ud-serve.sh) to serve the website locally.
